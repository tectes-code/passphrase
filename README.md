# Passphrase generator

A Passphrase is a secure password, made up of real words, which makes it quite easy to remember.  Most passphrase generators only allow to create passphrases with english words. This web app lets you choose between: German, English and a custom list of words.

Try it out at: https://tectes-code.github.io/passphrase/
